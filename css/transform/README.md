# Transform

## Syntax
* rotate(deg)
* rotateX(deg)
* rotateY(deg)
* rotateZ(deg)
* translate(x: px, y: px)
* translateX(x)
* translateY(y)
* translateZ(z)
* scale(x: 1.5, y: 1.5)
* scaleX(x)
* scaleY(y)
* scaleZ(z)
* skew(x: deg,y: deg)
* skewX(x)
* skewY(y)
